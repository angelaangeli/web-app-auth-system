# Web-приложение

**Описание**:

Проект состоит из двух сервисов:

1. Backend — веб-приложение для работы с файлами и базой пользователей(логин, пароль, ФИО, компания, должность)
2. Logger — микросервис для журналирования всех действий сервера и клиентов  в отдельную базу данных.

Все сервисы запускаются в Docker и используют PostgreSQL базы данных.


**Требования к работе с Web-приложением**:

На компьютере должны быть установлены:

1. Docker  
2. Docker Compose  


**Запуск web-приложения** :
1. Скопировать `.env-example` в `.env` и замениь значения на свои: `cp .env-example .env`

2. Сборка контейнеров:

В терминале в корневой папке проекта запустить команду `docker compose build`

3. Запуск приложения:
4. 
В терминале в корневой папке проекта запустить команду `docker compose up -d`


**Проверка работы web-приложения**:

Страница входа: http://localhost:8000/login

Страница загрузки файлов: http://localhost:8000/files

Страница с таблицей пользователей: http://localhost:8000/users

Страница микросервиса: http://localhost:9000/logs
